DispoLexi

Contraste de vocabulario.

El lenguaje es uno de los elementos que distinguen a los seres humanos del resto de los seres vivos, además permite la comunicación entre dos o más personas. El análisis del lenguaje ha tenido una gran relevancia en la historia del hombre, ya que puede ayudar a revelar aspectos y/o características importantes entre personas. DsipoLexi es una página web que busca ser una herramienta que ayude a los especialistas a obtener el léxico disponible y visualizar el contraste de vocabulario que hay entre diferentes grupos de interés a partir de los textos que estos producen.

Acerca de la Disponibilidad Léxica

La Disponibilidad Léxica (DL) es el campo de investigación que dentro de la lingüística tiene como objetivo la recolección y el posterior análisis del léxico disponible de una determinada comunidad de personas.

Prueba de Disponibilidad Léxica

La prueba de disponibilidad léxica se le aplica a un grupo de participantes hablan-tes. Se utilizan temas específicos sin ningún orden en particular (por ejemplo, salud,medicina, jardín, etcétera).

Cómo utilizar la aplicación web

La estructura del archivo es algo muy importante que se debe cumplir para poder utilizar la aplicaión web.

Estructura tu archivo archivo

El archivo tiene que estar en formato zip y debe contener carpetas (cada carpeta representa a una población a analizar), dentro de cada carpeta se deben colocar archivos de texto (cada archivo de texto representa una persona de la población) sólo en formato .txt y es importante no tener archivos fuera de las carpetas.

Carga tu archivo

Una vez que se tenga el archivo con la estructura seleccionada y el formato indicado, se debe cargar en la pagina de la aplicación y dar clic en 'subir archivo'

Selecciona las poblaciones

Si el archivo tiene la estrutura correcta se cargará y se mostrará una gráfica de pastel con las poblaciones que contiene el archivo, así como el número de archivos de texto por población. En este paso se debe elegir que poblaciones se desean contrastar (por defecto estarán todas las pobñaciones seleccionadas) y dar clic en procesar archivo.

Selecciona una gráfica

Después de que los archivos sean procesados, se mostrará una página donde se deberá elegir un tipo de gráfica para poder visualizar el contraste de vocabulario.

Configura la gráfica

Por último, se debe configurar la manera en que se desea visualizar el contrate de las distintas poblaciones. Se debe elegir las poblaciones a visualizar (por defecto estarán todas seleccionadas), se debe poner el número de palabras a visualizar y el tipo de palabras (con mayor o menor IDLV).
